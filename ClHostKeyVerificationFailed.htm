(Command line) Host key verification failed

 

When copying something from my Raspberry Pi to desktop computer, I got the following error:

 

 

My desktop computer has IP address: 888.188.88.888. The desktop computer is called the host. Regenerate the key for the host address (thanks to http://askubuntu.com/questions/20865/is-it-possible-to-remove-a-particular-host-key-from-sshs-known-hosts-file):

 

 

 

 

 

ssh-keygen -R 888.188.88.888
